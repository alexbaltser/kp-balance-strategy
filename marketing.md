# Маркетинговый план КП «Баланс»

---

## Бенчмарк: что делают топовые игроки

> Анализ лучших практик Sotheby's, Christie's, Knight Frank и ведущих luxury-девелоперов

### Ключевые принципы элитного маркетинга

1. **Визуальный сторителлинг** — не просто дом, а образ жизни
2. **Кинематографическое качество** — каждый кадр как произведение искусства
3. **Эмоциональная связь** — показать, каково жить в этом месте
4. **Иммерсивный опыт** — VR-туры, 360°, возможность "прогуляться" до покупки
5. **Мультиканальность** — адаптация контента под разные платформы
6. **Бренд и доверие** — репутация девелопера как ключевой актив

### Что используют топы

| Элемент                   | Описание                                                   |
| ------------------------- | ---------------------------------------------------------- |
| Фотореалистичные рендеры  | Высокая детализация, сложные материалы, разное освещение   |
| Кинематографическое видео | Дроны, закаты, lifestyle-сцены, профессиональный звук      |
| Виртуальные туры          | 3D walkthroughs, digital twins для удалённых клиентов      |
| Lifestyle-контент         | Не только дом, но виды, природа, инфраструктура, соседство |
| Бренд-стори               | История проекта, философия, ценности девелопера            |

## Необходимые материалы для запуска

### Рендеры экстерьера (10-12 штук)

| №   | Ракурс/вид                           | Назначение                           |
| --- | ------------------------------------ | ------------------------------------ |
| 1   | Главный фасад (дневной свет)         | Hero-изображение для сайта и рекламы |
| 2   | Главный фасад (вечер/закат)          | Атмосфера, премиальность             |
| 3   | Вид сзади / с террасы                | Показать приватную зону, бассейн     |
| 4   | **Общий вид посёлка сверху (дрон)**  | Масштаб, окружение, локация          |
| 5   | **Панорама посёлка с улицы**         | Как выглядит въезд и общая территория|
| 6   | Вид на виллу с ландшафтом            | Интеграция с природой                |
| 7   | Ночная подсветка                     | Архитектурная эффектность            |
| 8   | Панорама с видом на море/горы        | Показать виды, локацию               |
| 9   | **Въездная группа / КПП**            | Безопасность, первое впечатление     |
| 10  | **Общие зоны посёлка**               | Инфраструктура, благоустройство      |
| 11  | **Вид между виллами / улочки**       | Атмосфера соседства, приватность     |
| 12  | **Вид с участка на соседние виллы**  | Показать гармонию застройки          |

### Рендеры интерьера (6-8 штук на типовую виллу)

| №   | Помещение              | Особенности                     |
| --- | ---------------------- | ------------------------------- |
| 1   | Гостиная (общий вид)   | Масштаб, свет, отделка          |
| 2   | Кухня-столовая         | Техника, материалы, функционал  |
| 3   | Мастер-спальня         | Уют, вид из окна                |
| 4   | Мастер-ванная          | Премиальная сантехника, отделка |
| 5   | Терраса / веранда      | Indoor-outdoor living           |
| 6   | **Винный погреб**      | Ключевое УТП!                   |
| 7   | Вид из окна            | Панорама, природа               |
| 8   | Холл / лестница        | Архитектурные решения (опц.)    |

### Видеоконтент

#### Съёмка (видеограф делает исходники):

| №   | Что снимаем               | Описание                              |
| --- | ------------------------- | ------------------------------------- |
| 1   | Аэросъёмка локации (дрон) | Виды, панорамы, масштаб посёлка       |
| 2   | Стройка и территория      | Прогресс, качество работ              |
| 3   | Окрестности и природа     | Море, горы, инфраструктура            |
| 4   | Интервью с девелопером    | Если нужно — можно снять на месте     |

#### Монтаж (из исходников делаем ролики):

| №   | Итоговый ролик              | Хронометраж | Назначение                 |
| --- | --------------------------- | ----------- | -------------------------- |
| 1   | **Главный имиджевый ролик** | 2-3 мин     | Сайт, YouTube, презентации |
| 2   | **Обзор локации**           | 1-2 мин     | Показать место, виды       |
| 3   | **Тизер для соцсетей #1**   | 30-60 сек   | Анонс проекта              |
| 4   | **Тизер для соцсетей #2**   | 30-60 сек   | Второй вариант / формат    |

### Иммерсивные форматы

| Формат                           | Описание                             | Приоритет       |
| -------------------------------- | ------------------------------------ | --------------- |
| **360° панорама локации**        | Виды с участка, окрестности, природа | **Обязательно** |
| 360° панорамы территории посёлка | Въезд, общие зоны, улочки (5-7 шт)   | Высокий         |
| Интерактивный план               | Кликабельные зоны с фото             | Средний         |

## Чек-лист материалов для запуска

### Минимум для старта (Soft Launch)

- Рендеры экстерьера — 6-8 шт
- Рендеры интерьера — 4-5 шт
- Видеосъёмка (исходники)
- Главный видеоролик — 1 шт (монтаж)
- Тизеры для соцсетей — 2 шт (нарезка)
- Лендинг с формой заявки
- PDF-презентация для агентов
- PDF-презентация проекта (для клиентов)

### Полный комплект (Active Sales)

- Все рендеры экстерьера — 10-12 шт
- Все рендеры интерьера — 6-8 шт
- Дизайн-проект ремонта (2-3 варианта)
- Видеоролик + 2 тизера
- **360° панорама локации**
- Фото от видеографа
- Качественные PDF-презентации (3-4 версии)
- Полноценный сайт проекта
